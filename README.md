# Kenya 47 Counties - Cinematic 3D Experience

A cinematic, immersive 3D web experience that takes the viewer on a cultural journey across all 47 counties of Kenya.

## Tech Stack
- **Renderer**: Three.js
- **Animation**: GSAP + ScrollTrigger
- **Mapping**: D3-geo + Turf.js
- **Styling**: Vanilla CSS
- **Build Tool**: Vite

## Project Structure
- `/src/scenes`: Core Three.js scene logic
- `/src/shaders`: Custom GLSL shaders
- `/src/data`: GeoJSON and cultural data
- `/src/components`: UI components
- `/src/assets`: Textures and other static assets

## Development
```bash
npm install
npm run dev
```
